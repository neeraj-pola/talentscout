# Niche Rust / Embedded — Calibration Test

**This is the most important demo JD.** It probes whether the system
hallucinates strong matches when the pool genuinely doesn't have
qualifying candidates. Most LLM-based screeners fail this test —
they confabulate "this Python developer has some Rust on a side
project, so maybe..." A calibrated system says "I don't have anyone
for this."

red-flagged. This is correct behavior, not a system failure.
