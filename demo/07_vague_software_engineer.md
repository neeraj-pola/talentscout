# Vague Software Engineer

Underspecified JD resilience test. Recruiters routinely submit thin
descriptions like this. The system should still produce a sensible
generalist shortlist with moderate confidence — not silently fail or
produce falsely high scores.

Expected behavior: shortlist of 10 with the top candidate scoring
around 0.7-0.8, and many candidates flagged with must-have gaps
because the criteria are so vague.
